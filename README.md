## 应用说明

### 时刻台 · 关键事务提醒（index.html）
- 快速添加事务与闹钟提醒
- 按时间/优先级/标签分组查看
- 拖拽手动排序
- 标签多选筛选
- 闹钟弹窗 + 浏览器通知
- 重复提醒（每天/工作日/每周）
- 导入/导出 JSON 备份
- 深色/浅色主题切换

### 晷 · 事务提醒中心（任务提醒.html）
- 新建事务 + 优先级 + 分类
- 闹钟弹窗 + 稍后提醒
- 重复提醒
- 子任务管理
- 番茄钟专注
- 农历日期显示
- 导入/导出备份
- 深色/浅色主题切换
